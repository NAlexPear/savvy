JavaScript Building Blocks

  • Data Types:

    3 used in calculator.js

    - number

    - string ("hello world")

    - boolean (TRUE/FALSE)


  • Operators part 1:

    assignment

    - for variables: =, +=, -=, *=, /=, %=

We can do things to and with these data types by using operators. Operators allow us to manipulate data of certain types or allow us to extract information about those types. It’s how we make the sentences of programming. Most of the time, operators work best when you’ve saved the data you’re using in a variable.
Variables allow you to save data values (or functions, which we’ll see later), name them, and then bring them up later for use. You’ll be using a lot of variables as you start programming. Let’s make your first couple with some assignment operators. The most important one (=) simply assigns a value to a variable. Let’s try some of these:
  1. var myName = “John”;
  2. var myAge = 27;
  3. console.log(“Hi, my name is “ + myName + “ and I am “ + myAge + “ years old.”);

  • Operators part 2:

    data-type specific

    - numbers: +, -, *, /, %, ++, --

    - strings: +, +=

    - boolean: ===, !==, >, <, >=, <=, &&, ||

  1. console.log(myAge++);
  2. console.log(myAge);
  3. console.log(++myAge);
  4. myAge+=2
  5. var greeting = "Hello, my name is";
  6. console.log(greeting + “ “ + myName);
  7. var ageCheck = function(){if(myAge === 26){ console.log(“hooray!”); } else { console.log(“you done goofed”);};
  8. ageCheck();
  9. myAge += 3;
  10. ageCheck();
