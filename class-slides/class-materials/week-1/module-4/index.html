Exercise 3: Multiple Columns

A Page for Hipsters

  • Step 1:

    no layout

    -create a new HTML document

    -get some content from hipster ipsum

    -at least 10 paragraphs of hipster-y text!

  • Step 2:

    margin auto

    -create a fixed-width container div

    -center it using margin: 0 auto;

  • Step 3:

    brand and navigation divs

    -put all content inside div#content

    -div#content>div#header>div#nav>div#brand

    -put a logo and company name in div#brand

    -give divs different background colors

    -mock out some nav links

    -make nav links an unordered list

    -make nav links display horizontal (how?)

  • Step 4:

    fixed headers

    -try giving the header div position: fixed;


      • Answer the following:



    1. How do we prevent our header from overlapping content?
    2. What if the header is placed outside of the container?
    3. What if we give the header a set height and width?
    4. What happens when we mess with "top" and "left" values?

  • Step 5:

    a second column

    -create a new div#sidebar element

    -put some snazzy instapix in the sidebar

    -give div#container a rule of position: relative;

    -give div#content rules position: absolute; width: 600px;

    -give div#sidebar rules width: 350px; margin-left: 600px;

  • Step 6:

    a better way?



  • 
#content img {
  float: right;
  margin: 0 0 1em 1em;
}

  • try floating some images in div#content

  • Step 6:

    part 2!

    -remove position and margin-left properties from divs

    -add float: left to content and sidebar

    -add overflow: auto; to div#container

    -change the widths of content and sidebar to percentages!

    -try resizing the browser window

  • Step 7:

    final clean-up

    -add some padding, margins, borders to elements

    -don't forget about the border-box propety!
