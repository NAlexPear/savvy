  • Deploy your site:

    Firebase

    for easy CLI deployment of web applications


    1. Install node.js
    2. Sign up for Firebase (firebase.com)
    3. Use npm to install firebase globally

/c/SavvyCoders/community-page

$ npm install -g firebase-tools
    1. Authenticate your firebase CLI/initialize your project
    
$ firebase login
$ firebase init
  • Check out /html/firebase.json
    • 
{
  "firebase": "module-1-example-app",
  "public": ".",
  "ignore": [
    "firebase.json",
    "**/.*",
    "**/node_modules/**"
  ]
}

  • Deploy your new site:

    CLI only

    • 
$ firebase deploy
$ firebase open
